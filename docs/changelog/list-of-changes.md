# List of changes
{: .no_toc }

---

**New release 1.7.5 - 27.01.2020**
{: .label .label-purple }
```markdown
- improved logic for displaying additions if they don't contain identification
- improved settings - mostly translations of the texts
```

---

**New release 1.7.4 - 26.01.2020**
{: .label .label-purple }
```markdown
- added Stock management section - a new section for changing Stock state for any food with filtering and quick search functionality
- added Timeline section - a new section for reviewing the last actions, which the user made during working with the application
- added functionality to Skip to the start of the Grid/Column list by long-holding the left button
- improved Receipt design customization - changed the design for easier working with this screen
- improved Label design customization - changed design, added more text sizes for any info type
- improved information about results in the Orders History section/Stock Management section/Timeline section - this information will be displayed in the header under the title
- improved information in Food summary view - added additions, ingredients
- improved Food summary view logic - added highlight functionality even for the item header row
- fixed displaying items for Grid View/Columns View - fixed cutting the bottom part of the information during drawing the row, improved calculation logic
```

---

**New release 1.7.3 - 18.12.2019**
{: .label .label-purple }
```markdown
- added Columns view - almost same as Grid view, but every order is added below the previous one
- added Food Summary view - the list+grid content will be hidden
- added more text size options - increased option to set text size 3x more than default one
- added more color options - option to setup from a bigger palette of colors
- added option „Block Number In Front“ - number for every block will be displayed in the front of the content
- added option „Block Background Color“ - customize the color of the background for every block
- added option „Hide Title header“ - if the header is hidden, the only way to open the menu is swipe the left side to the right
- added option „Food Prefix For Deal, Item, Addition, Ingredient, Note“ - customize the text, that will be displayed before the particular order/food type
- added filtering options into Orders History screen - added filtering by payment type and changes (as the price changed, foods changed and so on)
- added option to select Default brand for filtering/Turn off filtering - you could filter the orders for default brand
- improved design for large screens (16'' and more) - adjusted design for correct displaying on the larger tablets after increasing text size option
- improved handling Receipt printer customization - added options 'Add empty lines at the start of the receipt', 'Add empty lines at the end of the receipt'
- improved redrawing of Grid view + Columns view - decreased memory consumption during the drawing of the rows
```

**New release 1.7.2 - 25.11.2019**
{: .label .label-purple }
```markdown
- added Food Summary view (needed to change List type in the settings)
- added option to finish all foods for the selected driver in Food Summary
- added Expand options for Food Summary view
- added option in setting „Turn Off Do Not Disturb mode“ if sounds are enabled (enabling this option in the Android will disable playing sounds, that's why this option was added)
- added Receipt printing for Star/BT/USB/Cashino printers
- added Orders history section
- added option to Print receipt/Kitchen receipt
- added option to preview receipt/kitchen receipt
- added option to manually reload Drivers list by swiping down the list
- added option for sorting food in Food Summary
- added option Alarm for new orders
- added option to switch order identifier in the list
- added option to Prepare/Finish order (changed logic of Prepare and Finish order, Prepare means setting all foods to the max for cooked/prepared, the finish will setup the order as finished)
- added option to Synchronize app settings through the account (only if specific account setting is enabled)
- moved Quick Collect option to set „List type“
- changed notification information about „No network connection“ into a circle in Header
```
