# Supported receipt printers
{: .no_toc }

---

### List of supported printers:
- **AURES** - USB
- **STAR** - Wi-Fi, Bluetooth, LAN
- **CASHINO** - Bluetooth, USB
- **BLUETOOTH** - Bluetooth
- **EPSON** - Wi-Fi, Bluetooth, USB
- **USB** - USB
